# Odin Admin Dashboard

An admin dashboard built with HTML and CSS Grid as part of [The Odin Project](https://www.theodinproject.com/) curriculum.

## Features

- **CSS Grid Layout** — Complex grid-based layout with 3 columns
- **Retro Theme** — Warm brown/red color palette
- **SVG Icon System** — External SVG sprite with Material Design Icons
- **Google Fonts** — Roboto typography

## Built With

- HTML5
- CSS3 (Grid, Flexbox)
- Material Design Icons
- Google Fonts (Roboto)

## Structure

```
odin-admin/
├── index.html          # Main dashboard page
├── style.css           # All styling
├── assets/
│   ├── svgs/           # SVG icon sprite
│   └── images/         # Avatar placeholders
├── docs/               # Reference project (gitignored)
├── AGENTS.md           # Agent instructions (gitignored)
└── README.md
```
